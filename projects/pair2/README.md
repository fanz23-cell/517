## Pair 2 Setup

This folder is for the second SO101 leader+follower pair.

Suggested ids:

- follower: `gix-follower2`
- leader: `gix-leader2`

Recommended workflow:

1. Configure follower motors.
2. Configure leader motors.
3. Calibrate follower with `gix-follower2`.
4. Calibrate leader with `gix-leader2`.
5. Save backup copies of the calibration JSON files into this folder.

Important:

- Keep only one controller board connected when finding a port or calibrating a single arm.
- Use `c` when `lerobot-calibrate` asks whether to run calibration again.
- Use the mechanical arm "0 position" from the course/Hugging Face docs.
